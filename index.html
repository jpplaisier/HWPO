"Flight Simulator"
-----
For Time:
5-10-15-20...45-50-45...15-10-5
Unbroken Double Unders *

* Rest as needed between sets.

Time Cap: 10 Minutes

NOTES:
- Be sure to respect the 10:00 time cap. The idea is that this will help manage volume and save the calves/ankles, especially if you are not yet very proficient with this movement.
- Each set must be unbroken. Once you complete a set, rest as much as you need to, then add five reps for another unbroken set. Do this until you get to 50 reps, then continue on in the same way, now decreasing by five reps each time.
- If you don't have double-unders, move to 5-10:00 of double-under practice.
